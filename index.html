Capabilities

能力矩阵

Planning

Task breakdown and progress-oriented agent workflows.

支持任务拆分与进度驱动的 Agent 工作流。

Filesystem + Exec

Structured file operations and command execution.

提供结构化文件操作与命令执行能力。

Sub-agents

Delegate scoped tasks to specialized agents.

将子任务委派给专用代理并隔离上下文。

Policy + Sandbox

Control risk with trust levels and policy gates.

通过信任等级与策略门控控制执行风险。

TUI + Headless

Run interactively or in script-friendly mode.

既可交互运行，也可脚本化无头执行。

Extension APIs

Compose with middleware, tools, skills, and adapters.

通过中间件、工具、技能与适配器自由扩展。
